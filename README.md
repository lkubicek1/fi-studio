# fi-studio

Minimal React + Vite project using a simple shadcn/ui setup.

## What is here

- `components.json` with a standard `new-york` shadcn configuration
- `src/components/ui/button.tsx` as the single reusable UI primitive
- `src/App.tsx` with a small button-driven counter demo
- `src/index.css` reduced to the core shadcn theme tokens and base styles

## Development

```bash
npm install
npm run dev
```

## Validation

```bash
npm run lint
npm run build
```
